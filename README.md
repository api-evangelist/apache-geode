# Apache Geode (apache-geode)
Apache Geode is an in-memory data management platform that provides real-time, consistent access to data-intensive applications throughout widely distributed cloud architectures. It pools memory, CPU, network resources, and local disk storage across multiple processes.

## APIs

### Apache Geode REST API
Geode provides a REST API for data access and cluster management, a Java client API for cache operations, Continuous Query API, Function Execution API, and OQL query language, plus .NET and C++ native clients.
